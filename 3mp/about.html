ABOUT

In these days of pandemic situation online delivery of items has taken a sudden rise and e-commerce business has been growing rapidly. An e-commerce website is a platform where customers can purchase items through online mode. So,the shopcart can be used to fulfill the need of an e-commerce website. Shop cart is an interface where users can buy day to day life products with a huge variety and can select all the desired items required, then add them collectively in the cart.

The price of all the products will be displayed and the final total can be seen at the time of confirmation of delivery. The skeleton of the website has been built using HTML, styling has been done using CSS and functioning of the website is done using Java script. In shopcart products can be added and removed easily based up on the stock available and many features can be seen like categorizing all the products into different departments, helpline, different modes of delivery and payment options.
